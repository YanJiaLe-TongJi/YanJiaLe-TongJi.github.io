模型蒸馏

我为什么开始认真记录模型蒸馏实验

做模型蒸馏时，我越来越强烈地感受到，真正重要的不只是最终分数，而是中间每一个选择为什么成立。学生模型结构、教师输出、损失权重、训练轮次，这些因素叠在一起时，直觉很容易失真。

所以我开始把每次实验的配置、假设和结果都认真记下来。这样的记录不只是为了复现，更是为了逐步形成自己的研究判断。

Agent Development

我最近为什么对 Agent Development 更感兴趣

Agent 最吸引我的地方，不只是“它能做点什么”，而是它如何进入真实环境之后依然能稳定工作。工具调用、上下文组织、失败恢复，这些问题都比表面的演示更有意思。

我希望把这些观察慢慢写出来，既整理自己的理解，也留下未来可以继续迭代的起点。

个人主页

为什么我决定用 GitHub Pages 搭自己的个人博客

我想要一个简单、稳定、长期属于自己的空间。GitHub Pages 对我来说足够轻，也足够直接，它不会把表达藏在平台规则后面。

这个博客会从一页开始，之后慢慢补上更多文章、项目和研究记录。它未必要一开始就很完整，但需要能够长期存在。
